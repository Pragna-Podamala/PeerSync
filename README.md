# 🚀 PeerSync – Real-Time Chat Application

PeerSync is a full-stack real-time messaging platform that enables seamless communication between multiple users with modern features and scalable backend architecture.

---

## 🌟 Features

- 💬 Real-time messaging using WebSockets (Socket.IO)
- 🔐 JWT-based authentication and secure APIs
- 👥 One-to-one and group chat support
- ❤️ Reactions and read receipts
- 📎 Media upload and sharing
- 🗳️ Polls and interactive messaging
- ⏳ Disappearing messages and scheduled messages
- 🛠️ Admin controls and user management

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- HTML5, CSS3

**Backend:**
- Node.js
- Express.js
- Socket.IO

**Database:**
- MongoDB

**Deployment:**
- Frontend: Vercel  
- Backend: Render  
- Database: MongoDB Atlas
